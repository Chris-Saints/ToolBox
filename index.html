Toolbox: Central Interativa

Um conjunto de ferramentas simples desenvolvidas com HTML, CSS e TypeScript. Este projeto é uma prática interativa para treinar fundamentos do Front-End.

Sobre o Projeto

Este Projeto foi criado com o objetivo de reunir e praticar conceitos básicos de desenvolvimento Front-End. Cada ferramenta foi feita com HTML, estilizada com CSS e traz lógica implementada com TypeScript.

A navegação é feita por páginas independentes, cada uma contendo uma funcionalidade interativa.

Ferramentas Disponíveis

  • Jogo do Clique
  • Formulário de Cadastro
  • Trocar Tema (Claro/Escuro)
  • Lista de Tarefas
  • Jogo das Cores
  • Relógio Digital
  • Calculadora de Soma
  • Mudar Cor de Fundo
  • Editor CSS
  • conversor de medida CSS
